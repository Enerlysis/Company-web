# Enerlysis website

Static landing page for Enerlysis.

## Files

- `index.html` — website content and structure
- `styles.css` — responsive visual design
- `script.js` — mobile menu behavior
- `CNAME` — custom domain for GitHub Pages
- `robots.txt` — search engine crawling settings
- `sitemap.xml` — basic sitemap for enerlysis.nl

## How to publish with GitHub Pages

1. Create a new GitHub repository, for example `enerlysis-website`.
2. Upload all files in this folder to the repository root.
3. Go to repository `Settings` → `Pages`.
4. Under `Build and deployment`, choose deployment from branch.
5. Select `main` branch and `/root` folder.
6. Add the custom domain `enerlysis.nl`.
7. In your domain/DNS provider, point the domain to GitHub Pages using the DNS records GitHub gives you.
8. Enable `Enforce HTTPS` when GitHub allows it.

## How to publish with Vercel

1. Go to Vercel and import the GitHub repository.
2. Keep it as a static project. No framework is required.
3. Add `enerlysis.nl` in Project Settings → Domains.
4. Update DNS records at your domain provider using the exact records Vercel shows.

## Before launch

Replace `info@enerlysis.nl` if you use another email.
Add your KvK number, VAT number, privacy policy and legal footer when ready.
